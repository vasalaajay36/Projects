# Vasala Ajay - Personal Portfolio

Welcome to my personal portfolio website repository! This project serves as a digital resume and a showcase of my frontend web development skills, demonstrating my ability to build elegant, responsive, and interactive digital experiences from scratch.

## 🚀 Live Demo
[View Live Site](https://vasalaajay36.github.io/Projects/)

## 🛠️ Built With

This project was built entirely without the use of heavy frameworks or external CSS libraries to demonstrate a strong foundational understanding of core web technologies:

*   **HTML5:** Semantic HTML markup for accessible and SEO-friendly structure.
*   **CSS3 (Vanilla):** Custom styling featuring modern techniques like CSS Grid, Flexbox, smooth transitions, and a responsive design that adapts to all screen sizes. Features custom gradients and subtle animations.
*   **JavaScript (ES6+):** Vanilla JavaScript used to handle DOM manipulation, smooth scrolling, and dynamic interactions without relying on jQuery or other DOM abstraction libraries.

## ✨ Key Features & Technical Highlights

*   **Responsive Design:** Fully responsive layout that ensures a seamless experience across desktop, tablet, and mobile devices.
*   **Modern Aesthetics:** Clean UI with a carefully selected color palette, modern typography (Inter & Outfit fonts from Google Fonts), and glassmorphism-inspired elements.
*   **Interactive Elements:** 
    *   Smooth scrolling navigation for better user experience.
    *   Dynamic background blob animation.
    *   Fade-in animations for content visibility as the user scrolls down the page.
*   **Performance Focused:** Lightweight implementation without external dependencies, resulting in lightning-fast load times.
*   **Accessibility (a11y):** Designed with proper contrast ratios and semantic tagging.

## 📁 Project Structure
